# Assignment 1: ServiceNow List & View Management

## Objective
To practice ServiceNow list personalization, filtering, creating custom views, and managing favorites to improve platform navigation efficiency.

## Tasks Performed
1. **List Personalization:** Configured the Incident list to display specific columns: *Number, Caller, Priority, Impact, and Urgency*.
2. **Advanced Filtering:** Used the Condition Builder to filter for "Critical Priority" incidents.
3. **Custom View Creation:** Created a new view named **"Demo View"** with columns: *Number, Caller, Short Description, and Priority*.
4. **Favorites Management:** Saved the filtered lists as Favorites for quick access from the sidebar.

## Key Learnings
- Difference between Personalizing a list (Gear Icon) and Configuring List Layout (View).
- How to use the Condition Builder to generate complex queries.
- Managing navigation efficiency using Favorites.
